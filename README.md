# [SuiSwapX Sui SDK]

## Functionality

- swap
- findPoolId
- createPool

## Install

Install the project dependencies with yarn or npm

```bash
  yarn add @sui-swap-x/sui-sdk
```

## Requirements

Node.js version >= 18

### [Install node.js LTS](https://nodejs.org/en)

Verify your Node.js version with 

```bash
  node -v 
```
